Added `classify_tag_role()` to `versioning.py` to classify any managed tag as 'exact' or 'floating' from the local inventory alone (with cross-prefix isolation). Added `sync_all()` to `tag_service.py` as a pure remote-conformance operation that iterates all local managed tags and applies the same push safety rules as version-scoped push. Updated `cli.py` to make `<version>` optional (`nargs="?"`), route versionless `--push` to `sync_all()`, handle `version_str: str | None` in output helpers, and reject `--switch` without a version. Added 39 new tests across all four test files. Bumped to v0.4.0.
